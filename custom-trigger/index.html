How to use? ▼

You can filter by:

  • group - test.describe("test_login",...) => {"group": {"eq": {"test_login"}}}
  • id - allure.label("id", "22") => {"id": {"eq": "22"}} Parametrized tests have the same id!
  • name - test full name => {"name": {"eq": "test_get_partners_negative"}}
  • labels - allure.epic("API") => {"labels": {"epic": {"eq": "API"}}}

Filter supports operators:

  • eq, neq - value equal or not equal.
  • in - matches tests where the field contains at least one of the specified values.
  • nin - matches tests where the field contains none of the specified values.
  • all - matches tests where the field contains every specified value.
  • exists - boolean condition.

Filter strategies:

  • {key:value, key:value, ...} - filtered result by one condition with multiple params
  • [{key:value}, {key:value, key:value, ...}, ...] - filtered result + filtered result +...

You can write filter query without "" and ''. Like {id:{eq:22}} Hope it works well!
